Aether lets you build HTML components directly in Python — no templates, no string concatenation. You compose elements using Python's call syntax, pass typed attributes, and render clean HTML output. It integrates with HTMX, Alpine.js, Tailwind CSS, and Chart.js without leaving Python.

## Why Aether?

Aether gives you the full power of Python — loops, conditionals, functions, and classes — to build HTML components without ever leaving your Python workflow.

## Get started

<Steps>
  <Step title="Install Aether">
    ```bash
    pip install pyaether
    ```
  </Step>
  <Step title="Import elements and render">
    ```python
    from aether.tags.html import H1, Div, P
    from aether import render

    page = Div(_class="container")(
        H1()("My Awesome Page"),
        P()("This is a paragraph of text.")
    )

    print(render(page))
    ```
  </Step>
  <Step title="Explore the docs">
    Check out the [Quickstart](/quickstart) for a full walkthrough, or browse the [Core Concepts](/concepts/elements) to understand how Aether works.
  </Step>
</Steps>
