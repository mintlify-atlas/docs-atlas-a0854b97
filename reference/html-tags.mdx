`aether.tags.html` provides a Python class for every standard HTML5 element. Each class accepts element-specific keyword arguments and inherits the common global attributes from `GlobalHTMLAttributes`. Import the elements you need directly from the module.

```python
from aether.tags.html import Div, P, H1, Button, Form, Input, Span, A, Img
```

## Global attributes

Every HTML element class inherits from `GlobalHTMLAttributes`, which defines the attributes common to all HTML elements.

<Note>
  Use `_class` in Python to set the HTML `class` attribute — the leading underscore is stripped at render time. Underscores in any attribute name are converted to hyphens (e.g., `data_value` → `data-value`).
</Note>

Common global attributes include: `id`, `_class`, `style`, `hidden`, `role`, `tabindex`, `lang`, `title`, `aria_label`, `data_*`, and all other [ARIA attributes](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Attributes).

Each element also has a matching `*Attributes` TypedDict (e.g., `DivAttributes`, `ButtonAttributes`) that documents its element-specific attributes.

See [elements](/concepts/elements) and [attributes](/concepts/attributes) for more on how attributes work.

## Document structure

Elements that form the skeleton of an HTML document.

| Element | Attributes TypedDict | Notes |
|---|---|---|
| `Html` | `HtmlAttributes` | Root element |
| `Head` | `HeadAttributes` | Document metadata container |
| `Body` | `BodyAttributes` | Document content container |
| `Title` | `TitleAttributes` | Document title (text only) |
| `Meta` | `MetaAttributes` | Metadata (void element) |
| `Link` | `LinkAttributes` | External resource link (void element) |
| `Script` | `ScriptAttributes` | Inline or external script |
| `Style` | `StyleAttributes` | Inline CSS |
| `Noscript` | `NoscriptAttributes` | Fallback for no-script environments |

```python
from aether.tags.html import Html, Head, Body, Title, Meta, Link, Script, Style, Noscript

page = Html(lang="en")(
    Head()(
        Meta(charset="utf-8"),
        Title()("My page"),
        Link(rel="stylesheet", href="/styles.css"),
    ),
    Body()(
        Script(src="/app.js"),
    ),
)
```

## MathML

Aether also provides a `aether.tags.mathml` module for embedding MathML. The current release exports one top-level element:

| Element | Attributes TypedDict | Notes |
|---|---|---|
| `Math` | `MathAttributes` | Root MathML element |

```python
from aether.tags.mathml import Math

Math(display="block")("MathML content")
```

`MathAttributes` accepts `display` (`"block"` or `"inline"`, defaults to `"inline"`). Additional MathML child elements are not yet exported — use the `Math` container to wrap raw MathML markup via `mark_safe` if needed.
